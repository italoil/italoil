# ITALOIL Shopify Theme - Deployment Checklist

Use this checklist when deploying the ITALOIL theme to production.

## Pre-Deployment

### Theme Files
- [ ] All theme files are committed to repository
- [ ] .gitignore excludes unnecessary files
- [ ] No sensitive data in code (API keys, passwords)
- [ ] All files use UTF-8 encoding

### Testing Locally
- [ ] Theme validated with Shopify CLI (if used)
- [ ] All Liquid syntax checked
- [ ] CSS validated
- [ ] JavaScript error-free
- [ ] No console errors

## Shopify Store Setup

### Store Configuration
- [ ] Shopify plan selected and active
- [ ] Domain configured (or using .myshopify.com)
- [ ] Store timezone set correctly
- [ ] Currency set to EUR (â‚¬)
- [ ] Store language set to English

### Legal Pages
- [ ] Privacy Policy created
- [ ] Terms of Service created
- [ ] Refund Policy created
- [ ] Shipping Policy created
- [ ] Contact Information page created

### Shipping
- [ ] Shipping zones configured
- [ ] Shipping rates set up
- [ ] International shipping options (if applicable)
- [ ] Free shipping thresholds (if any)

### Payments
- [ ] Payment provider connected (Shopify Payments, PayPal, etc.)
- [ ] Test payment completed successfully
- [ ] Payment currency verified (EUR)

## Theme Upload

### Upload Process
- [ ] Theme packaged as ZIP (excluding .git folder)
- [ ] Uploaded to Shopify admin
- [ ] Upload completed without errors
- [ ] Theme appears in theme library

### Initial Configuration
- [ ] Logo uploaded
- [ ] Brand colors verified (#2C5530, #F5F1E8, #C8A882)
- [ ] Fonts selected (Playfair Display, Lato)
- [ ] Favicon uploaded
- [ ] Social media links added
- [ ] Contact email updated to italoil.store@gmail.com
